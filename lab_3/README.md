«Комплексный анализ и прогнозирование многомерных временных рядов метеопараметров»

## 1. Исходные данные
В работе используются данные метеорологических наблюдений (температура, осадки и др. параметры телеметрии) для шести городов, относящихся к различным климатическим зонам.
Файл data.zip

## 2. Задание

### 2.1. Разведочный анализ данных (EDA)
* **Визуализация:** Анализ трендов, суточной и годовой сезонности, идентификация выбросов и пропусков.
* **Распределения:** Построение гистограмм, box-plot и Q-Q plot для проверки нормальности признаков.
* **Корреляции:** Анализ взаимосвязей между метеопараметрами с помощью тепловых карт.
* **Кросс-городской анализ:** Сравнение климатических профилей городов (среднемесячные температуры, осадки).
* **Стационарность:** Проверка рядов тестом Дики-Фуллера и применение дифференцирования.

### 2.2. Этап 1: Классификация
* Определение типа климата на основе временного ряда метеопараметров.
* Каждый тип климата представлен минимум двумя городами.

### 2.3. Этап 2: Прогнозирование
* Построение моделей прогнозирования температуры на горизонт **H = 30 дней**.
* **Используемые подходы:**
    * Статистические модели и градиентный бустинг (например, CatBoost).
    * Глубокое обучение (Temporal Fusion Transformer).
    * Гибридные архитектуры на базе обучения на остатках (Residual Learning).
* Обоснование выбора архитектуры для различных климатических зон.

### 2.4. Оценка качества и анализ
* **Метрики:** MAE, RMSE, MAPE, WAPE для каждого города.
* **Анализ остатков:** * Тест Шапиро-Уилка (нормальность).
    * Тест Льюнга-Бокса (автокорреляция).
    * Тест Уайта (гетероскедастичность).
* **Визуализация результатов:** Сравнение прогнозов с фактом, анализ распределения ошибок (box-plot).
